Take credit card payments on your Wordpress site using Stax Payments.

== Description ==

Integrate Stax Web Payments with your WordPress website as easily as adding a button.

Just add your Web Payments Token and optionally customize your button to fit your brand. Add it to posts, pages, or any other content via the included Gutenberg block or shortcode generator, and you're ready to start accepting payments!

When a user clicks your button, the Stax checkout will be launched seamlessly in a pop-up on your website.

== Installation ==

1. Upload `stax-payments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or simply go to 'Plugins' > 'Add New' in the WordPress admin and search for "Stax Payments" and install/activate it from there.

== Frequently Asked Questions ==

= How do I get Web Payments Token? =

Login to your Stax account and click on <strong>"Apps"</strong>, then <strong>"Website Payments"</strong>, then <strong>"Launch"</strong>, or just follow the link below: <a href="https://app.staxpayments.com/#/apps/webpayments" alt="Get your Web Payments Token from Stax Application" target="_blank">https://app.staxpayments.com/#/apps/webpayments</a>
